2.4.0

Изменен принцип установки value и id из индексированных массивов т.е. [1,2] значения будут 1 и 2 сответсвенно, а не 0 и 1 как раньше. Чтобы использовать число в качестве value отличное от title, необходимо в массиве конуретно указать значение key. Например ["40 " => test] (обратите внимание на пробел). Из-за того что php преобразует строки, содержащие целое число к int, приходится добавлять пробел либо в начало, либо в конец ключа. В итоге пробелы в начале и в конце удаляются автоматически.
